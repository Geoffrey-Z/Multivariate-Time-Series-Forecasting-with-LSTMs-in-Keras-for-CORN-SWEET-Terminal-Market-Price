# Multivariate-Time-Series-Forecasting-for-Corn-Sweet-Terminal-Market-Price-with-LSTMs-in-Keras

This project implements a sweet corn termical market price forecasting using LSTMs in Keras by Weiling(Geoffrey) Zhu.

## Report

The report for this project can be viewed [here](https://github.com/Geoffrey-Z/Multivariate-Time-Series-Forecasting-with-LSTMs-in-Keras-for-CORN-SWEET-Terminal-Market-Price/blob/master/Report.md).

## Prerequisites

This project uses Google Colab (https://colab.research.google.com/).

This project is written in Python 3.
